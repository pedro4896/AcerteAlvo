  • Este é um código JavaScript que simula um jogo de tiro ao alvo. A função principal do jogo é disparar no alvo, que aparece aleatoriamente na tela. O código usa a API Canvas do HTML5 para desenhar o alvo e interagir com o usuário. O jogo tem cinco níveis de dificuldade e cada nível apresenta diferentes valores de velocidade, erros, pontuação e quantidade de munição disponível para o jogador.
